15세기 사대부 한시 교류 네트워크

651명 저자 중 생몰년이 15세기(1400~1499)에 걸치는 인물 중 실제 교류가 매칭된 100명 간의 증답 관계만 추려낸 네트워크입니다. 세종~성종대 관각문인·사림파 1세대가 중심입니다. 원본은 651개 문집 시 362,130수에서 규칙 기반으로 추출·매칭한 결과라 노이즈가 섞여 있습니다.

– 인물(교류 있음)
– 저자→수신자 관계
15세기 (1400–1499) 필터
651명 전체 결과는 별도 네트워크 참고
저자 수신자 분류 횟수 시대 관계 저자 생몰 수신자 생몰
읽는 법: 실선 = 생몰년이 겹치는 동시대 교류. 점선(회색) = 동시대가 아님(대개 ‘차운’으로 후대 사람이 옛 시인의 운을 빌려 지은 것 — 실제 왕래가 아닌 문학적 오마주). 굵은 빨강 점선 = 저자가 수신자보다 먼저 사망 — 시간상 불가능하므로 인물 매칭 오류일 가능성이 높습니다.
조작: 노드에 마우스를 올리면 연결 관계가 강조되고, 드래그로 위치를 옮길 수 있습니다. 그래프가 뭉쳐 보이면 “간격” 슬라이더로 펼친 뒤, 마우스 휠로 축소하거나 더블클릭해 전체 보기로 돌아오세요.
한계: 이 필터는 651명 전체 결과(위 네트워크와 동일한 매칭 로직)에서 생몰년만 15세기로 추려낸 것이라, 전체 네트워크와 같은 매칭 한계(14.4% 매칭률, 모호 매칭 등)를 그대로 가지고 있습니다.
